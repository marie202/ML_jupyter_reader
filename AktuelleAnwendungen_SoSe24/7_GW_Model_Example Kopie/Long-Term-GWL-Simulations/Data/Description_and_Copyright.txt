#############################################
Weekly groundwater level time series dataset for 118 wells in Germany
#############################################

This data is published with kind permission of the local state authorities mentioned below.
When using this dataset, please be sure to appropriately identify the data sources and include copyright statements, as shown below.
For more information on the sites please search in the primary data sources below or check the paper referenced here: https://github.com/AndreasWunsch/Long-Term-GWL-Simulations 

Based on previous knowledge from similar/surrounding time series (Wunsch and Liesch 2020) some  included time series were interpolated.
Please find the respective information on interpolated values in the folder "Interpolations".
Red dots indicate interpolation based on previous knowledge, blue dots indicate pchip interpolation.
Time series not mentioned in this folder do not contain any interpolated values.

References:
Wunsch, A. & Liesch, T. Entwicklung und Anwendung von Algorithmen zur Berechnung von Grundwasserständen an
